## Порядок использования

1. Не начинать большой VST3 до завершения Phase 0 и script MVP.
2. В конце каждого этапа применять gate из roadmap.
3. Если gate не пройден, менять гипотезу или прекращать ветку, а не скрывать проблему дополнительными функциями.
4. Любое расширение scope сначала добавляется в backlog, а не автоматически в текущий sprint.
