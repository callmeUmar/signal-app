# Landing

План публичной маркетинговой поверхности Signal.

## Роль

Landing не является просто красивой обложкой. Он должен:

- быстро объяснить отличие Signal от генераторов готовых песен;
- показать реальный workflow `MIDI context → candidates → editable result`;
- дать попробовать ограниченный browser demo;
- привести к регистрации/dashboard;
- объяснить различие browser и FL sidebar;
- сформировать доверие через privacy, control и provenance;
- дать download/install путь после появления sidebar beta.
